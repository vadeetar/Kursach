# Как оформить курсовую в Word

## Полная курсовая (рекомендуется)

```powershell
cd E:\курсач
py -3 -m pip install python-docx
py -3 docs\generate_kursovaya_full.py
```

Файл: **`E:\курсач\KURSOVAYA_RABOTA.docx`** — титул, задание, реферат, все главы, приложения.

## Краткая записка

```powershell
py -3 docs\generate_docx.py
```

Файл: **`POYASNITELNA_ZAPISKA.docx`**

## Какую модель AI выбрать в Cursor

См. **`docs/KAKOY_MODEL_VYBRAT.md`** — для текста лучше **Opus 4.8 High** + **MAX Mode**.

## Шаг 2 — Доработать в Word

1. **Титульный лист** — замените строку про вуз на официальный шаблон вашей кафедры (логотип, полное название МГТУ/института).
2. **Оглавление** — Вставка → Оглавление → Автособираемое (заголовки 1–2 уровня).
3. **Нумерация страниц** — с 2-й страницы (после титула): Вставка → Номер страницы.
4. **Рисунки** — вставьте скриншоты вместо текстов `[Рисунок N ...]`:
   - Рис. 1 — архитектура (можно экспорт из Mermaid или схема в Paint)
   - Рис. 2 — дашборд http://127.0.0.1:8000
   - Рис. 3 — Swagger /api/v1/docs
   - Рис. 4 — GitHub Actions (зелёный CI)
5. **Подписи** — под каждым рисунком: «Рисунок 1 — Архитектура системы» (по центру, Times 12).
6. **Подписи в конце** — ФИО, дата, подпись руководителя.

## Шаг 3 — Типичные требования ГОСТ (проверьте у кафедры)

| Параметр | Значение в DOCX |
|----------|-----------------|
| Шрифт | Times New Roman, 14 пт |
| Интервал | 1,5 |
| Поля | левое 3 см, правое 1,5 см |
| Абзац | отступ 1,25 см |
| Выравнивание | по ширине |

## Шаг 4 — Скриншоты

См. `SCREENSHOTS_GUIDE.md` — запустите `.\start.ps1`, сделайте Win+Shift+S.

## Шаг 5 — Сохранить для сдачи

- **POYASNITELNA_ZAPISKA.docx** — для правок
- **POYASNITELNA_ZAPISKA.pdf** — Файл → Сохранить как PDF (часто сдают PDF)

## Если python-docx не ставится

1. Откройте `POYASNITELNA_ZAPISKA.md` в VS Code / Cursor.
2. Скопируйте в Word.
3. Выделите заголовки `##` → стиль «Заголовок 1», `###` → «Заголовок 2».
4. Таблицы в markdown вставьте вручную или через Pandoc:

```powershell
pandoc POYASNITELNA_ZAPISKA.md -o POYASNITELNA_ZAPISKA.docx
```
