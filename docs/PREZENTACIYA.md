## Слайд 2 — Актуальность

- Рост числа CVE (>20 000 в год)
- Зависимости PyPI/npm — основной вектор риска
- Нужна автоматизация: сбор → приоритизация → исправление

## Слайд 3 — Цель и задачи

**Цель:** автоматизировать управление уязвимостями в ПО организации.

**Задачи:**
1. Интеграция с NVD и EPSS  
2. Инвентаризация пакетов  
3. Сопоставление CVE с активами  
4. Приоритизация (CVSS + контекст)  
5. Веб-интерфейс статуса исправления  
6. Docker, тесты, CI/CD
